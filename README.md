# ag-news-topic-classification
A complete end-to-end NLP pipeline for classifying news articles into four categories (World, Sports, Business, Sci/Tech) using a fine-tuned Hugging Face BERT model (bert-base-uncased). Built with PyTorch and the AG News dataset, achieving 90.8% test accuracy. Includes a live interactive web UI deployed with Gradio for real-time inference.
